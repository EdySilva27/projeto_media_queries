Testando Media Queries

"Uma media query consiste de um media type e pelo menos uma expressão que limita o escopo das folhas de estilo usando media features, tal como largura, altura e cor. Media queries, adicionadas no CSS3, deixam a apresentação do conteúdo adaptado a uma gama especifica de dispositivos não precisando mudar o conteúdo em si." developer.mozilla.org Acesso em Nov. de 2022< https://developer.mozilla.org/pt-BR/docs/Web/CSS/Media_Queries/Using_media_queries > Acesso em Nov. de 2022
